# Algo-Deck Backend v1.0

## Backend for the Algo-Deck platform, built with NestJS, Prisma, and Clerk. Handles authentication, user management, and secure API endpoints.

### Features

<ul>
<li>Token-based authentication with Clerk</li>
<li>User creation and retrieval</li>
<li>Frontend-secret middleware for restricted access</li>
<li>Rate limiting with NestJS Throttler</li>
<li>PostgreSQL database managed with Prisma</li>

</ul>
